# План управління проєктом
## Intelligent Currency Tracker

---

## 1. Загальна інформація про проєкт

**Назва проєкту:** Intelligent Currency Tracker  
**Мета:** Створення системи для моніторингу та прогнозування валютних курсів у реальному часі з використанням AI-моделей  
**Дата початку:** Січень 2025  
**Дата завершення:** Листопад 2025  
**Власник проєкту:** Андрій Алмаші  
**Статус:** Завершено

---

## 2. Цілі та завдання

### Основні цілі:
1. Забезпечити точне відстеження валютних курсів у реальному часі
2. Реалізувати прогнозування курсів за допомогою машинного навчання
3. Створити інтуїтивний інтерфейс для аналізу тенденцій
4. Автоматизувати систему сповіщень про ризики

### Ключові показники успіху (KPI):
- Точність прогнозів > 85%
- Час відгуку API < 500ms
- Доступність системи > 99%
- Покриття тестами > 80%

---

## 3. Архітектура та технології

### Backend:
- .NET 9.0, ASP.NET Core Web API
- Entity Framework Core, PostgreSQL
- Kafka для обробки потоків даних
- JWTAuthentification, Identity

### Frontend:
- Blazor WebAssembly
- Razor Components

### Data Science:
- Python (Pandas, Scikit-learn, TensorFlow)
- Аналіз часових рядів

### DevOps:
- Docker, Docker Compose
- Nginx (reverse proxy)
- CI/CD pipeline
